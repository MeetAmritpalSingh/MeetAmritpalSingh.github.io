Claim Mapping to Workspace Format Converter

Upload an Excel file to convert claim mapping data into a workspace-friendly format, then download the transformed Excel file.

Important Notes
  • Rows above the first identified data row will be treated as taxonomy nodes.
  • The first data row is detected where column A contains a patent-number style identifier.
  • When claim mappings contain multiple values, separate them with |.
  • The first worksheet in the workbook will be used.
  • Do not include columns where text details are needed, such as Relevant Text, Additional Claim Limitations, or Others node.
  • You may still need to adjust and merge headers in the downloaded Excel sheet.
