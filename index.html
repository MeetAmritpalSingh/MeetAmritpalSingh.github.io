Claim Mapping to Workspace Format Converter

Upload an Excel file here. It will convert the given Excel into a workspace-friendly format and download the transformed Excel output.

Important Notes
  • Rows above the first identified data row will be considered as taxonomy nodes.
  • The first data row is detected where column A contains patent number format.
  • When claim mappings contain multiple values, they should be separated by |.
  • The first worksheet in the workbook will be used.
  • Do not include columns where text details are needed, for example, Relevant Text, Additional Claim Limitations, or Others node.
  • There may be a need to adjust and merge headers in the downloaded Excel sheet.
